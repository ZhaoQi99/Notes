# Notes
Backup for Obsidian notes.


## How to use
* Download [Obsidian](https://obsidian.md/download)
* `git clone git@github.com:ZhaoQi99/Notes.git`
* `git config user.name "Qi Zhao";git config user.email "zhaoqi99@outlook.com"`
* Open Obsidian -> 打开本地仓库 -> Notes

## Optional
* `CMD + P`
	* `Git: open history view`
	* `Git: Open source control view`
